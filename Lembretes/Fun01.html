                                      (vetor, string, tamanho da string) 

                        void string(char *pont, char *nome,int qnt){ 
                            int i;
                            for(i=0; i < qnt ; i++){
                                pont[i] = nome[i];
                            }
                        };

Exemplo: (01/janeiro/2000) terceira posição do vetor é j, ele irá verificar a terceira posição do vetor, a variável ind recebe 3 porque na situação do exemplo ele verifica a partir do 3° elemento do vetor

Caso verifique apenas o mês, (janeiro), a variável ind será 0

                    Se começar com J, pode ser janeiro,junho ou julho
                    Se segundo tiver o elemento a, é janeiro
                    Senão
                        Se terceiro elemento for n, é junho
                        Senao, é julho
